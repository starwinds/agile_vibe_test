# Product Requirements Document (PRD)

## 개요
이 프로젝트는 PostgreSQL 16 + pgvector 확장 기능을 활용하여 Python 애플리케이션에서 문장 임베딩 데이터를 저장하고, 코사인 유사도 기반으로 검색하는 샘플 앱을 구축하는 것을 목표로 합니다.

## 목표
- PostgreSQL과 pgvector를 사용한 벡터 데이터베이스 구축
- Python을 이용한 문장 임베딩 및 저장
- 저장된 임베딩을 활용한 유사도 검색 기능 구현
- TDD 방식의 개발 프로세스 적용

## 사용자 시나리오
- 개발자는 텍스트를 입력하여 임베딩 벡터로 변환하고 데이터베이스에 저장할 수 있다.
- 개발자는 쿼리 텍스트의 임베딩과 가장 유사한 데이터를 데이터베이스에서 검색할 수 있다.

## 기능 정의
- `embedding.py`: 텍스트를 임베딩 벡터로 변환하는 기능
- `db_utils.py`: 데이터베이스 연결, 테이블 생성, 데이터 삽입 및 검색 기능
- `app.py`: (추후 개발) Flask 기반의 웹 인터페이스 제공
